# [RD-VBAL]: VBA Language Platform Specification
---

> ℹ️ **RD-VBA** is an implementation of the MS-VBAL specification that is independent from its historical **MS-VBA** runtime host;  
> **RD-VBAL** is the name of the specification/documentation of the _language server platform_, which _includes_ the RD-VBA _language core_.
